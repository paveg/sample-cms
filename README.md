## Quick Start

- Please fork this repository.
- And, pre-execution `npm install` .

### next

```bash
$ npm run dev
# Launch server http://localhost:3000
```

### with vercel

```bash
$ npm install -g vercel
$ vercel dev
# Launch server http://localhost:3000
```
